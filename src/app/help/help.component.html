See Car List

Visit the home page to see the list of cars available. This list will update as new cars will be added or exitsing car will be deleted.

The list has an action column that contains button for edit and delete.


Delete a Car

Clicking on delete button will show a pop-up. If delete delete action is confirmed then car will be deteleted.


Add New Car

Visist the 'Add Car' page to open the form. Fill the form and submit it. If any input is empty, form will not submit. After form submits, application will redirect to list page.
